# Модуль Network Data

## Описание
Модуль network-data предназначен для получения информации о всех подсетях в выбранной VPC в Yandex Cloud.

## Функционал
- Получает информацию о выбранной VPC по ID
- Получает полный список всех подсетей в VPC
- Возвращает детальную информацию о каждой подсети

## Входные переменные
- `vpc_id`  - VPC ID для получения информации о подсетях

## Выходные данные
- `vpc_id` - VPC ID
- `vpc_name` - Имя VPC
- `subnets_info` - Информация о всех подсетях
- `zones`  - Список всех уникальных зон с подсетями
- `subnets_count` - Количество подсетей в VPC
